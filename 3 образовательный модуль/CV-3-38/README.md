# Система распознавания ценников с проверкой на подделку

## Описание проекта

Проект реализует систему компьютерного зрения для анализа изображений ценников.  
Алгоритм определяет, является ли ценник **оригинальным (бумажным)** или **поддельным (отображённым на экране)**, а также извлекает **итоговую сумму** с помощью OCR.

---

## Основная задача

1. Проверить текстуру ценника: оригинальный (бумага) vs подделка (экран)
2. Извлечь цену
3. Если текстура не соответствует — вывести статус **«подозрение»**
4. Вывести итоговый результат

---

## Baseline и метрики

- **Baseline**: подделка обнаружена  
- **Метрика**: корректная классификация (paper / screen)
